# MySQL Datenbank
In diesen Ordner soll alles bezüglich der MySQL DB rein. Hiergehört:
- *.sql Dateien um die DB erstellen zu können
- ERM- oder UML-Diagramm zum Aufbau der DB

Am besten benutzt jeder xampp um sich lokal eine DB anlegen zu können und HeidiSQL um ein gutes DBMS zu haben.
	- Download Link für xampp: https://www.apachefriends.org/de/index.html
	- Download Link für HeidiSQL: https://www.heidisql.com/download.php (portable)
	- Link zum DB-Designer: https://dbdesigner.net/

Wir können uns überlegen ob wir die DB auch bei einem externen Hoster anlegen.
